## 1️⃣ Tokenization - التقطيع

**ما هو؟**  
تقطيع النص إلى وحدات صغيرة (كلمات أو جمل).

**الكود المستخدم:**
```python
from nltk.tokenize import word_tokenize
text = "الولد يأكل التفاحة"
tokens = word_tokenize(text)  # ['الولد', 'يأكل', 'التفاحة']
```

**التطبيق في مشروعنا:**
- تقسيم التغريدات إلى كلمات مفردة
- تحضير النصوص للمراحل التالية
- حساب عدد الكلمات في كل نص

**النتيجة:**
```
قبل: "المنتج رائع جداً"
بعد: ['المنتج', 'رائع', 'جداً']
```

---

## 2️⃣ Stopwords - كلمات التوقف

**ما هو؟**  
إزالة الكلمات الشائعة التي لا تحمل معنى مهم.

**الكود المستخدم:**
```python
from nltk.corpus import stopwords
stop_words = set(stopwords.words('arabic'))
words = [w for w in tokens if w not in stop_words]
```

**التطبيق في مشروعنا:**
- إزالة كلمات مثل: في، من، على، إلى
- تبقى الكلمات المهمة: رائع، سيء، ممتاز
- تحسين دقة النموذج

**النتيجة:**
```
قبل: ['المنتج', 'رائع', 'جداً']
بعد: ['المنتج', 'رائع']
```

---

## 3️⃣ Stemming - الاشتقاق

**ما هو؟**  
تحويل الكلمات إلى جذورها الأصلية.

**الكود المستخدم:**
```python
from nltk.stem.isri import ISRIStemmer
stemmer = ISRIStemmer()
root = stemmer.stem("يكتبون")  # 'كتب'
```

**التطبيق في مشروعنا:**
- توحيد الكلمات المختلفة لنفس الجذر
- يكتب، كاتب، مكتوب ← كتب
- تقليل عدد الكلمات الفريدة

**النتيجة:**
```
يكتب ← كتب
كاتب ← كتب
مكتوب ← كتب
```

---

## 4️⃣ POS Tagging - وسم أجزاء الكلام

**ما هو؟**  
تحديد نوع كل كلمة (اسم، فعل، صفة، حرف).

**الكود المستخدم:**
```python
# قاموس بسيط
pos_dict = {
    'محمد': 'اسم',
    'يأكل': 'فعل',
    'جميل': 'صفة'
}
```

**التطبيق في مشروعنا:**
- فهم تركيب الجمل
- التركيز على الصفات (مهمة لتحليل المشاعر)
- تحسين دقة التصنيف

**النتيجة:**
```
محمد ← اسم
يأكل ← فعل
التفاحة ← اسم
في ← حرف جر
```

---

## 5️⃣ NER - التعرف على الكيانات

**ما هو؟**  
استخراج الكيانات المهمة من النص (أشخاص، أماكن، منظمات).

**الكود المستخدم:**
```python
import re
names = re.findall(r'محمد|أحمد|فاطمة', text)
```

**التطبيق في مشروعنا:**
- استخراج أسماء الأشخاص
- تحديد الأماكن المذكورة
- فهم سياق النص بشكل أعمق

**النتيجة:**
```
النص: "محمد يعيش في القاهرة"
الكيانات: 
   👤 محمد → شخص
   🏙️ القاهرة → مكان
```

---

## 6️⃣ Word Embedding - تمثيل الكلمات

**ما هو؟**  
تحويل الكلمات إلى متجهات رقمية تفهم العلاقات بينها.

**الكود المستخدم:**
```python
from gensim.models import Word2Vec
model = Word2Vec(sentences, vector_size=100)
vector = model.wv['كلب']  # [0.2, -0.5, 0.8, ...]
```

**التطبيق في مشروعنا:**
- فهم التشابه بين الكلمات
- كلب ≈ قط (حيوانات)
- سيارة ≠ كلب (مختلفان)

**النتيجة:**
```
كلب ← [0.9, 0.8]  
قط  ← [0.8, 0.7]  (قريبان)
سيارة ← [0.1, 0.9] (بعيد)
```

---

## 7️⃣ TF-IDF - وزن الكلمات

**ما هو؟**  
إعطاء وزن للكلمات حسب أهميتها في النص.

**الكود المستخدم:**
```python
from sklearn.feature_extraction.text import TfidfVectorizer
vectorizer = TfidfVectorizer()
X = vectorizer.fit_transform(texts)
```

**التطبيق في مشروعنا:**
- الكلمات النادرة تحصل وزن أعلى
- "رائع" في نص واحد مهم
- "المنتج" في كل نص غير مهم

**النتيجة:**
```
الكلمة     الوزن
رائع       0.8  (مهمة)
المنتج     0.1  (شائعة)
سيء        0.7  (مهمة)
```

---

## 8️⃣ LDA - تحليل المواضيع

**ما هو؟**  
اكتشاف المواضيع المخفية في مجموعة النصوص.

**الكود المستخدم:**
```python
from sklearn.decomposition import LatentDirichletAllocation
lda = LatentDirichletAllocation(n_components=3)
lda.fit(X)
```

**التطبيق في مشروعنا:**
- تصنيف التغريدات لمواضيع
- الموضوع 1: جودة المنتج
- الموضوع 2: خدمة العملاء
- الموضوع 3: سعر المنتج

**النتيجة:**
```
الموضوع 1: [رائع، ممتاز، جيد، حلو]
الموضوع 2: [سيء، بطيء، غالي، خربان]
الموضوع 3: [رد، اتصال، شكوى، مساعدة]
```
